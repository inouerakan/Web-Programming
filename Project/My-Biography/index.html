Tentang Saya

Nama lengkap saya Rakan Shaka Raufa biasa dipanggil Rakan. Saya lahir di Garut, 18 November 2008. Saya adalah anak pertama dari dua bersaudara. Saya senang belajar hal baru, terutama di bidang teknologi dan informasi. Saya mulai memasuki jenjang pendidikan pada usia 4 tahun, yaitu di TK Kadeudeuh. Saya menempuh TK selama dua tahun. Setelah SD, saya melanjutkan ke MTS Sirnamiskin. Di MTS saya mengikuti ekstrakurikuler komputer sebagai wadah untuk menyalurkan minat di bidang teknologi dan informasi. Kemudian saya melanjutkan ke SMK Negeri 4 Bandung. Di SMK, saya diterima di program keahlian Rekayasa Perangkat Lunak (RPL). Saya mengikuti ekstrakurikuler Icon Jurnalistik, di ekskul tersebut saya aktif sebagai anggota divisi editor. Selama mengikuti ekstrakurikuler Jurnalistik, saya beberapa kali mendapatkan penghargaan sebagai Best Member bulanan di divisi editor.
